Installation Guide


Step1: 
      Go to your Private-Eye_linux_packaging folder

Step2:
	Open your Terminal and Type these commands

	
	$ chmod +x cfile && chmod +x install && chmod +x uninstall
	$ ./install


Now you can find your application in your applications list
